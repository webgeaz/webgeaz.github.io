    Google Guava Formatting Utilities

    Guava provides useful tools and utilities for formatting text in a consistent and readable manner. This utilities is simplifying the string creation, structured outputs, and error messages.

    1. Joiner

    Imports Statements:

    from com.google.common.base import Joiner

    ðŸ”¹Joining Strings and Collections:

    Useful for joining list elements into a single string with a delimiter.

    Usage Examples:

    items = ["apple", "banana", "cherry"]
joined = Joiner.on(", ").join(items) # Output : apple, banana, cherry

    ðŸ”¹Skiping Nulls:

    Nulls are ignored, unlike Python's native join() which fails with None.

    Usage Examples:

    items = ["apple", None, "banana", "cherry"]
joined = Joiner.on(" | ").skipNulls().join(items) # Output : apple | banana | cherry

    ðŸ”¹Replacing Nulls

    Replaces nulls with a specified string.

    Usage Examples:

    items = ["apple", None, "banana", "cherry"]
joined = Joiner.on(", ").useForNull("N/A").join(items) # Output : apple, N/A, banana, cherry

    2. Splitter

    Imports Statement:

    from com.google.common.base import Splitter

    ðŸ”¹Splitting Strings into Lists:

    Trims spaces and splits strings into elements using a delimiter.

    Usage Examples:

    text = "apple, banana, cherry"
items = Splitter.on(", ").trimResults().splitToList(text) # Output : [apple, banana, cherry]

    ðŸ”¹Splitting with Custom Delimiters:

    Filters out empty strings from the result.

    Usage Examples:

    text = "apple,,banana,,cherry"
split = Splitter.on(",").omitEmptyStrings().splitToList(text) # Output : [apple, banana, cherry]

    ðŸ”¹Key-Value Mapping:

    Creates a dictionary-like map from a string (great for config parsing).

    Usage Examples:

    text = "x=1;y=2;z=3"
split_map = Splitter.on(";").withKeyValueSeparator("=").split(text) # Output : {x=1 , y=2, z=3}

    3. Case Format Conversion

    Convert between naming styles: camelCase, snake_case, UPPER_CASE, etc.

    Import Statement:

    from com.google.common.base import CaseFormat

    Usage Examples:

    camel = "userNameField"
converted = CaseFormat.LOWER_CAMEL.to(CaseFormat.UPPER_UNDERSCORE, camel) # Output : USER_NAME_FIELD

    4. CharMatcher â€“ Filtering and Transforming Strings

    Keeps only the characters matching a condition (e.g., digits, letters, whitespace).

    Import Statement:

    from com.google.common.base import CharMatcher

    Usage Examples:

    text = "abc123XYZ"
filtered = CharMatcher.digit().retainFrom(text) # Output :123

    5.Strings.padStart / padEnd

    Pads a string from the start or end with a specific character.

    Import Statement:

    from com.google.common.base import Strings

    Usage Examples:

    padded = Strings.padStart("42", 5, "0")  # 00042
