    Google Guava I/O Utilities

    Guava simplifies I/O with Files, ByteStreams, and CharStreams, supporting file and stream operations in fewer lines.

    1. Reading a File as Lines

    Import Statements:

    from com.google.common.io import Files
from java.io import File
from java.nio.charset import Charset

    Usage Examples:

    file = File("example.txt")
lines = Files.readLines(file, Charset.forName("UTF-8"))

    Note

    Reads all lines from a file into a list of strings.

    2. Writing to a File

    Import Statements:

    from com.google.common.io import Files
from java.io import File
from java.nio.charset import Charset

    Usage Examples:

    file = File("output.txt")
Files.asCharSink(file, Charset.forName("UTF-8")).write("Hello from Guava!")

    Note

    Writes a full string to a file using a CharSink.

    3. Read Entire File as a Single String

    Import Statements:

    from com.google.common.io import Files
from java.io import File
from java.nio.charset import Charset

    Usage Examples:

    file = File("example.txt")
text = Files.asCharSource(file, Charset.forName("UTF-8")).read()

    Tip

    Reads the full content of a file into a single string.

    4. Copy File Content

    Import Statements:

    from com.google.common.io import Files
from java.io import File

    Usage Examples:

    source = File("source.txt")
destination = File("copy.txt")
Files.copy(source, destination)

    Note

    Copies the contents of one file to another.

    5. Create Temporary Directory

    Import Statements:

    from com.google.common.io import Files

    Usage Examples:

    temp_dir = Files.createTempDir()
print("Temporary dir created:", temp_dir.getAbsolutePath())

    Tip

    Creates a unique temporary directory. Ideal for intermediate processing.

    6. Get File Extension and Name Without Extension

    Import Statements:

    from com.google.common.io import Files
from java.io import File

    Usage Examples:

    file = File("archive.tar.gz")
ext = Files.getFileExtension(file.getName())  # "gz"
name = Files.getNameWithoutExtension(file.getName())  # "archive.tar"

    Note

    Extracts file extension or name without extension from a File.

    7. Touch a File (Like UNIX touch)

    Import Statements:

    from com.google.common.io import Files
from java.io import File

    Usage Examples:

    # Specify your folder path and filename
folder_path = "my_data/logs"
file_name = "touched_file.txt"

# Create the folder if it doesn't exist
folder = File(folder_path)
if not folder.exists():
    folder.mkdirs()

# Create the File object pointing to the desired path
file = File(folder, file_name)

# Use Guava to touch the file
Files.touch(file)

    Tip

    Creates the file if it doesn't exist, or updates last modified time if it does.

    8. Read Text from Resource File (Classpath)

    Import Statements:

    from com.google.common.io import Resources
from java.nio.charset import Charset

    Usage Examples:

    url = Resources.getResource("email_template.html")
content = Resources.toString(url, Charset.forName("UTF-8"))

    Note

    Loads and reads a file from the classpath—useful for reading bundled config or templates.
