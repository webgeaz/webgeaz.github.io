    SUFIA Administration¶

    SUFIA Administration modules includes functionality to administer SUFIA and the application build on SUFIA.

    Some functionality is for application development (File Manager, Settings), while others are commons functions that can be used by application build on SUFIA (and SUFIA itself).

    Functionality like user management, role management and menu management is commons for most web application. You can use these function as is or modify it according to your application needs (SUFIA API). By using the ready made function, you can speed up your application development process.

    Administration functions require user under category System Management.
