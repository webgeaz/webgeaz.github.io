    Upload / Download¶

    Upload¶

    To make Transaction support upload file, you need to create a field using Form Builder with the type BLOB. Give the variable name for the field.

    In your Fom Template, add input type file with name attachments[0].

    Warning

    This feature is using convention, you must follow the guideline in order to make it works

    At the Post-Processor, you can just use the variable name to save the uploaded file. SUFIA will generate another 2 fields automatically for file name and file type:

    For variable name file_upload →

    • file_upload - byte array (content of file)

    • file_upload_fn - file name

    • file_upload_ft - file type

    Example (Form Template)

    <form action="/t/HR/report?invoke" method="post" enctype="multipart/form-data">
    <div class="form-group">
        <input type="file" name="attachments[0]">
        <button type="submit" name="d_action" value="upload">
          Upload
        </button>
    </div>
</form>

    (Form Builder)

    (Post-Processor)

    import Report_attc

if d_action == "upload":
    if file_upload:
        report_attc = Report_attc()
        report_attc.set("file", file_upload)
        report_attc.set("file_fn", file_upload_fn)
        report_attc.set("file_ft", file_upload_ft)
        report_attc.saveIt()

    Download¶

    To download file, you either can use StreamingResolution (refer here) or use the build in helper servlet:

    <img src="/bl?tn=report_attc&cn=file&k=1"/>
    • tn - table name
    • cn - column name
    • k - row id (key)
    • il - inline (e.g. il=1 for inline)
