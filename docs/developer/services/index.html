    Services - SUFIA API¶

    SUFIA provides API in order to access the internal classes of SUFIA. This is useful if you need to use or override the default behavior of certain aspects of SUFIA e.g. login process, user creation etc.

    SUFIA API can be invoked using the SUFIA pre-defined variables: service

    UserService¶

    MailService¶

    package sufia

    Class MailService

    This class provides access to mail services.

    Method Summary

    • public void send(String from, String[] to, String[] cc, String[] bcc, String title, String message, String[] attachments, boolean isHtml)

    Parameters:

    • from - email address of the sender
    • to - array of email addresses for recipient
    • cc - array of email addresses for cc
    • bcc - array of email addresses for bcc
    • title - subject for email
    • message - content of email
    • attachments - array of attachments file path
    • isHtml - true if content is in HTML format or false (plain text)
