    Handlebars Helper¶

    SUFIA additional helper for Handlebars. You may need to refer to Handlebars documentation to understand the concept.

    Value Lookup¶

    Sometime you want to display the name / description of a particular id. For instance, consider you have table hr_department_staff with column staff_id, which is a foreign key to table hr_staff. While iterating the hr_department_staff records, you may want to display the staff name instead of the staff_id.

    Usage:
{{ref_lookup ref_id table="tableName" label="name" value="id"}}

    Example: 

    <ul>
    {{#each staffs}}
        <li>{{ref_lookup staff_id table="hr_staff" value="id" label="name"}}</li>
    {{/each}}
</ul>

    HTML format¶

    Transform newline to <br> and make valid URL clickable.

    Usage:
{{html context}}

    Example: 

    <p>{{html content}}</p>

    Generate HTML <select> tag¶

    Generate HTML <select> tag with data from database table. This helper support parent-child selector (e.g. Country & State)

    Usage:
{{select models? selected=context? table="tableName"? filter="type=1"? id="id"? name="name"? class="class"? childIds=""? parent=""? label="name" value="id"}}

    Example: 

    {{select table="hr_staff"}}

    Generate HTML <input>,<textarea> tag¶

    Input tag helper to cater for read-only value. If view is true input is automatically disabled.

    Usage:
{{input value? ...}}
{{textarea value? ...}}

    Example: 

    {{input staff.name class="form-control" name="name" maxlength="150" view=view_only}}

    Date Formatting¶

    Default formatting of date to dd/MM/yyyy

    Usage:
{{dateFormat context "pattern"?}}

    Example: 

    <p>{{dateFormat today}}</p>

    Contains Helper¶

    Check if the value exists in the given context. Works with array or list

    Usage:
{{#in context value}}...{{/in}}

    Example: 

    {{#in admins staff_id}}
    <p>This user is an Administrator</p>
{{/in}}

    Equals & Not Helper¶

    Check if the value if equals or not equals.

    Usage:
{{#eq value1 value2}}..{{/eq}}
{{#not value1 value2}}..{{/not}}

    Example: 

    {{#eq staff.sex "1"}}
    Male
{{else}}
    Female
{{/eq}}

    Length, Size¶

    Get the length of list, arrays

    Usage:
{{length context}}

    Example: 

    <p>Total users: {{length users}}</p>
